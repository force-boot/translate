# translate
调用方式:[GET|POST]  

请求参数：  

content： 必填 翻译内容  

to： 选填 翻译目标语种 默认en  

from： 选填 当前源语言 默认auto 指定可提高准确性  

dataType: 必填 数据格式 默认json 可选 string  

请求示例:  

GET：http://translate.xxxxx.cn/translate/json/?content=你好&to=en&from=auto  

POST：http://translate.xxxxx.cn/translate/json/
